# Bloco de Notas

Um editor de texto simples feito em C# Windows Forms durante as aulas do curso no Senac Santos em 2024.

## Sobre

O projeto foi desenvolvido em sala de aula. A ideia era criar um bloco de notas parecido com o do Windows, colocando em prática o que estava sendo aprendido sobre Windows Forms e manipulação de arquivos em C#.

## O que ele faz

- Criar um novo arquivo (com confirmação antes de limpar o conteúdo)
- Abrir arquivos de texto
- Salvar o conteúdo em arquivo
- Alterar a fonte do texto
- Tela "Sobre" com informações do projeto

## Tecnologias

- C#
- .NET Windows Forms
- Visual Studio

## Como rodar

1. Clone o repositório
2. Abra o arquivo `bloco de notas 2.sln` no Visual Studio
3. Pressione F5 para executar
